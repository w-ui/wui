# WUI

WUI是一个基于vue的移动端组件库，是移动端开发的抽象和沉淀，是提升移动开发效率的神器。


# 特色

|                          |                               |                            |
|:------------------------:|:-----------------------------:|:--------------------------:|
| 高覆盖 <br>50+组件         |  高质量 <br>90%测试覆盖率       |  简单易用 <br>可视化辅助编辑   |
| 高可定制 <br>支持自定义样式  |  体积小 <br>严格优化按需加载     |  高性能 <br>大量级数据检验     |
